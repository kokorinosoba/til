# About Tag

```html
<starttag>...</endtag>
<element property=""><element>
```

# Tags

```html
<h1>Heading</h1>

<p>Paragraph</p>

<br> Line Break

<a href="#">Anchor</a>

<img src="Source" alt="Alternate">

<ul>
  <li>Unordered List</li>
</ul>

<ol>
  <li>Ordered List</li>
</ol>

<table>
  <tr>
    <th>Heading1</th>
    <th>Heading2</th>
  </tr>
  <tr>
    <td>Data1</td>
    <td>Data2</td>
  </tr>
</table>
```

# Shorthand for Tag Writing

```html
ul>li*6
<ul>
  <li></li>
  <li></li>
  <li></li>
  <li></li>
  <li></li>
  <li></li>
</ul>

table>(tr>th*2)+(tr>td*2)*2
<table>
  <tr>
    <th></th>
    <th></th>
  </tr>
  <tr>
    <td></td>
    <td></td>
  </tr>
  <tr>
    <td></td>
    <td></td>
  </tr>
</table>
```
