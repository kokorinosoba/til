# First JavaScript

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>First JavaScript</title>
  </head>
  <body>
    <script>
      document.write('1 plus 2 is');
      document.write(1 + 2);
    </script>
  </body>
</html>
```

# Divide HTML and JavaScript

```html
<!-- my-first-javascript.html -->
<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>First JavaScript</title>
  </head>
  <body>
    <script src="myFirst.js"></script>
  </body>
</html>
```

```js
// myFirst.js
document.write('1 plus 2 is');
document.write(1 + 2);
console.log('Result displayed.');
```
